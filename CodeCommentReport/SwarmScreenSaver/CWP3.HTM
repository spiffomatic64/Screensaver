This class contains the info on the swarms, the state of the swarm, and the wasp and the
intitialization routines and drawing methods to run this screensaver to any Graphics object passed in.

Access: Public
Base Classes: Object
  Members Description  
    StopNow This is used for the caller to inform Insects that it needs to stop now!
This is used in IF and For statements to help this class stop drawing
when the screen is to be destroyed.

  
    eraseOldLines    
    Bees Default value is a count of 144. Count of bees. This is configureable. Defualts are required here
because the program cries foul when I pass them to methods, it thinks I'm going to try to access an
unassigned variable. Pah!

 
    BeeVel Default Bee Velocity although this is recalculated later to be 92% of overall velocity.
This is configureable.

 
    ColorCycleSeconds How many seconds before the swarm color changes. Default:1 Configureable.

 
    GlitterModeWanted Whether Glotter is wanted. Default is false.
Drawing is done to a in memory bit map and then is copied to the screen. No flicker, faster drawing!

 
    screenHeight height of drawing area in pixels.

 
    screenWidth Width of drawing area in pixels.

 
    Border Wasp is not allowed any closer to edge of screen than this value.
This is to reduce the swarm going off screen.

 
    WaspOldX    
    WaspOldY    
    WaspX Wasp X position as a three int element array. I don't know why 3 elements.

 
    WaspY Wasp Y position as a three int element array. I don't know why 3 elements.

 
    WaspVel Default wasp max velocity. Configureable. It is compared against the two X and Y
velocity of the Wasp and the bees.

 
    WaspXVel    
    WaspYVel    
    dtColorCycleSecondsTime Used to determine if it's time for the swarm to change color. First run through, we add -1 variables to
thia variable, to make it PAST time to pick a color. I don't want to always start with red.

 
    TIMES I think there could have been a better name for this variable but such is the original code.
First Rank for x and y in swarmstruc. Ex: x[Times,Bee#] = SomeVelocity;

 
    BeeAccMax Ohhh some constants. I wish steady work was a constant for me instead
of jumping from one Titanic to another.
Maximum accelleration for bees. A speed limit for the byte cops!

 
    WaspAccMax Maximum Wasp Accelleration. Accelleration is not to be confused with Velocity.

 
    SwarmPen This pen is for drawing the swarm. Second parameter is the width of the line in pixels.
Default color is red.

 
    WaspPen    
    BlackPen    
    commFunctions An instance of the class CommonFunctions. The random number and color contrast functions are here.
The contrast functions are used to ensure that the swarm not too dark to see.

 
    swarm Creates ONE swarm of bees to flee. Ouch!!

 
    clsInsects    
    ~clsInsects Created a destructor to rid the SwarmPen and commFunctions.

    SetSwarmPenToRandRgbColor Gets a random color into the SwarmPen.Color.
then calls AdjustForeColorBrightnessForBackColor to brighten up invisible colors.
